### 高速ゼータ変換
 - N×2^Nに計算量を圧縮
 - 以下どっちでも同じ
   - if (i & 1 << bit) != 0 { a[i] += a[i & !(1 << bit)]; }
   - if (i & 1 << bit) == 0 { a[i | 1 << bit] += a[i]; }
   - [二項係数](arc137/src/bin/d.rs)
